# 작업 기억 용량의 한계 (Working Memory Limits)

> **Seed note.** 이 노트는 Phase 2에서 Core 프로토콜의 근거 링크 타겟으로 생성된 시드입니다. 내용은 Research Integration Protocol을 통해 점진적으로 채워집니다.

## 한 줄 요약

인간의 작업 기억 용량은 약 4±1 청크(Cowan의 현대적 수치)이며, 이 한계가 한 번에 비교할 수 있는 후보 수·검토할 수 있는 제약 수·유지할 수 있는 부분 해를 제약한다.

## 핵심 개념 (Key concepts)

- Cowan의 4±1은 Miller의 고전적 7±2를 간섭과 리허설을 통제한 실험으로 갱신한 현대 수치다.
- Chunking(청크화)은 의미 단위로 정보를 묶어 유효 용량을 확장하는 주요 우회 전략이다.
- Sweller의 인지 부하 이론(cognitive load theory)은 본질적·외재적·유관 부하로 구분하며, 외재적 부하 감소가 학습과 문제 해결을 촉진한다.
- 체계적 외재화(paper/file/whiteboard)는 작업 기억 한계를 우회해 복잡한 비교와 부분 해 유지를 가능하게 한다.

## 6단계 프로토콜에서의 역할

모든 단계의 체크리스트 항목 수 상한(3–5 또는 3–7)이 이 한계에 맞춰 설계됨. Converge의 '≤ 3 surviving candidates'도 동일 근거.

## 추가 연구 필요

- 전문가-초보자 청크 크기 차이와 도메인별 확장 정도
- 수면 부족·스트레스 상태에서의 작업 기억 저하 정도
