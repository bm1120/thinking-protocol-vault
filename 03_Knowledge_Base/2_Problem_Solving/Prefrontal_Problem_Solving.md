# 전전두엽의 문제 해결 역할 (Prefrontal Problem Solving)

> **Seed note.** 이 노트는 Phase 2에서 Core 프로토콜의 근거 링크 타겟으로 생성된 시드입니다. 내용은 Research Integration Protocol을 통해 점진적으로 채워집니다.

## 한 줄 요약

전전두엽(PFC)은 문제 해결의 네 가지 기능을 담당한다: 목표 표상(dlPFC), 오류 감지(ACC), 주의 제어(lateral PFC), 억제 제어(vlPFC·IFG).

## 핵심 개념 (Key concepts)

- dlPFC(dorsolateral PFC)는 현재 목표를 표상하고 작업 기억을 유지하며, 하위 단계의 진행을 감독한다.
- ACC(anterior cingulate cortex)는 예측과 결과의 불일치를 감지해 오류 신호를 발생시킨다.
- Lateral PFC는 과제 관련 정보에 주의를 집중시키고 방해 자극을 억제하는 주의 조절 역할을 수행한다.
- vlPFC·IFG는 응답 억제(response inhibition)를 담당하며, 이 기능의 효율은 유동 지능(fluid intelligence)과 강한 상관을 보인다.
